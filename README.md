# Macroeconomic-Comparative-Analysis-India-vs-USA-
Macroeconomic Comparative Analysis: India vs USA | Under Dr. Neelanjan Datta, IIT Kanpur

• Performed a comparative macroeconomic analysis of India and the United States using historical data from
1950–2025 across GDP components, inflation, unemployment, labour share, and per capita growth.

• Developed 10+ time-series visualizations in Python using Pandas, Matplotlib, and Seaborn, highlighting the 1973
Oil Crisis, 2008 Global Financial Crisis, and COVID-19 pandemic.

• Derived insights on growth volatility, structural transformation, labour-capital income distribution, and macroe
conomic resilience in developing versus developed economies
